# 东京购房攻略2:通向地主之路
# 什么是収益Myhome
简单来讲就是除了自住部分外 </br>
还带了出租部分的Myhome, </br>
需要自己买地，找建筑公司设计施工。</br>
看过穷爸爸富爸爸的朋友 </br>
应该都知道负债与资产的概念。</br>

只是单纯的买了房 </br>
并不能简单的算作资产，</br>
而能每月带来安定收入的収益 </br>
不折不扣的属于资产。 </br>

简单的算一笔账，</br>
如果収益Myhome的每月房租收益部分能 </br>
覆盖贷款且有5万剩余，</br>
首先节省了每月房租支出，</br>
假设一个月10万一年就是120万, </br>
房贷退税每年30+万, </br>
出租-贷款每月剩余5万, </br>
每年就能多出来200万的可支配收入。 </br>

不知道各位工资多少，</br>
按30%的税率如果要增加200万的可支配收入，</br>
那么至少要账面上的收入要涨300万，</br>

另外这笔收入你还可以用于再投资，</br>
如此好事何乐而不为呢？ </br>

接下来介绍一下収益Myhome的好处和坏处 </br>

## 好处
* 可以用住房贷款
  * 更低的利率，更长的还款年数 
* 自住部分能降低空室率风险
* 同住一楼管理租客方便

## 坏处
* 需要有买地的眼光
  * 需要能预见什么的地能盖什么样的房
  * 周边环境是否是和出租，租金多少
* 工期
  * 和现成的商品房不同建成前无法确认
  * 买完地后就要开始还贷初期压力大
* 一直和租客做邻居
  * 邻里关系要处理好

如果这些坏处你都能接受的话， </br>
那就接着往下看怎么实际操作吧。 </br>

# 如何选地
* 最基础条件
  * 必须是建筑用地 
  * 建筑基本发的道路规定
  * 不是[借地]
* 过于便宜的土地容易有坑
  * 旗竿敷地
  * 周围环境不好

房子的地点决定了它的资产价值，</br>
一分钱不得马虎，</br>
地点决定了出租部分的空室率，</br>
地点决定了之后如果要出手是否有买家解盘。 </br>
所以多走多看多思考。</br>
关于地点，可以参考上一篇 [东京购房攻略](https://github.com/cheungYX/algorithm/blob/master/cheung/ff.md)</br>


## 容积率与建蔽率
容积率就是所有楼层的总面积占土地面积的比率， </br>
建蔽率是和土地接触部分的面积占土地面积的比率， </br>
一块地的容积率与建蔽率限制了能盖什么养的房子。 </br>

需要注意的是建筑地和面向的道路的接触部分要超过2m， </br>
以及各种水道电线杆之类的细节规定， </br>
很多具体细节请查看具体建筑地所在城市的规划局， </br>
不过放心，卖房的中介都有义务说明这些。 </br>

## 自住部分的面积觉得建筑面积
首先先考虑自己想要住多大面积的房子， </br>
一般来说2LDK50平，3LDK75平。 </br>
决定了自住部分的面积， </br>
基本也就决定了出租部分的面积了。 </br>

面积和你所能贷到的资金， </br>
决定了你大概房子能盖在哪个地段。 </br>

## 如何找地
* 登陆各种网站
* 新闻广告
* 现地不动产公司
* [建築条件付き]的物件
  * 简单来说就是设计和施工公司已经定好
  * 虽然建设费用不好控制，但土地价格会比市面便宜

# 如何找建筑公司盖房
## 工法与建筑成本
* 木造 50万円/坪
* S造(铁骨) 60万円/坪
* RC造(钢筋混凝土) 70万円/坪

木造住宅的建筑费50万里， </br>
大概原材料费大概30万+， </br>
扣去人工和设备费用剩下的就是建筑公司的利益。 </br>
如果对材料有要求， </br>
或者要追加个床暖房建筑成本还会再上升。 </br>

## 增加面积压低成本的技巧
* 增加面积的技巧
  * ロフトの設置
  * 隣地との隙間を活用
  * 階段持ち上げ
  * 水回りをコンパクト
* 缩减成本的技巧
  * 水道負担金
  * プロパンガスを導入
  * 出租部分的设备可以偏简陋

## 出租部分的户型
* 参考周边平均租金
* 同等户型周边供给是否过剩
* 计算概算收支并考虑空室风险

可以先要多个小户型的预算报价  </br>
比如3个1k，然后要去改成2个稍微大的， </br>
并要去降价

# 整体流程
总体来说，比起现成的房子， </br>
収益Myhome的周期要长，手续要多  </br>
建筑期间你还得同时附带房租和房贷， </br>
前期的经济和心里压力都会比较大， </br>
需要提前做好储备。 </br>
我们来回顾一下整体流程  </br>
* 1 计划立案
  * 确定贷款额度
  * 确保头金
  * 确定自住房的必备条件
* 2 找地
  * 收集情报
  * 现地视察
  * 周边市价调查
  * 向不动产公司请求计划书
  * 确认收支预算
  * 最终确认，契约
* 3 设计，建筑
  * 设计计划的具体化
  * 开工(4个月到半年)，做好延期的资金准备
  * 交房，登记
* 4 交房，招募
  *  招募(完工前一个月开始募集)
  *  租房管理

* 资金流程
  * 土地契約時 手付金
  * 土地決済時 残金
  * 建物本契約時 手付金,着手金
  * 建物上棟時 中間金
  * 建物決済時 残金
